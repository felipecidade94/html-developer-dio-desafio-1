História da computação

(Voltar)

A história recente da computação, desde primeiros computadores até o uso da inteligência artificial (IA)

As 5 gerações dos computadores

Os primeiros computadores

Os primeiros computadores eram máquinas enormes que ocupavam salas inteiras e tinham capacidade de processamento muito inferior aos nossos smartphones atuais. Os computadores modernos são muito mais rápidos e compactos, graças ao avanço da tecnologia.
  1. A 1ª geração de computadores é composta pelo Mark I (em 1944) e o Colossus (1946) e o Eniac (1946)
  2. A 2ª geração (1959 até 1965) corresponte ao uso de transistores
  3. A 3ª geração (1965 até 1970) corresponde ao uso de circuitos integrados
  4. A 4ª geração (1971 até ~2000) é caracterizada pela popularização dos microprocessadores e dos computadores pessoais. Foi nessa geração que houve o surgimento da internet
  5. A 5ª geração (dias atuais) é caracterizada pela popularização da internet e dos dispositivos móveis, internet das coisas (IoT), computação em nuvem inteligência artificial (IA)

Interação Cliente x Servidor

O que são clientes

Os clientes são as máquinas que acessam os servidores para obter informações.

O que são servidores

Os servidores são máquinas que armazenam informações e as disponibilizam para os clientes.

O que é HTML

(Voltar)

O HyperText Markup Language (HTML), ao contrário do que alguns pensam, não é uma linguagem de programação.

HTML é usado para descrever a estrutura e o conteúdo de uma página da web. Ele define elementos como títulos, parágrafos, links, imagens, etc., mas não inclui lógica ou funcionalidade interativa.

Diferente de linguagens de programação, como JavaScript, Python ou Java, HTML não suporta lógica de programação como loops, condições ou funções. Ele não executa nenhuma ação, apenas organiza e exibe conteúdo.

O HTML é estático por natureza. Ou seja, ele exibe o conteúdo de forma fixa. Para criar interatividade ou dinâmica em uma página web, você precisa usar linguagens de programação junto com HTML, como JavaScript.

HTML é uma linguagem de marcação que usa tags para estruturar o conteúdo de uma página web. As tags são palavras-chave cercadas por colchetes angulares, como <tag>. Cada tag HTML descreve diferentes tipos de conteúdo na página.

Tags em HTML

As tags em HTML são usadas para formatar o texto e adicionar imagens e links.

Existem várias tags em HTML. As aprendidas até aqui nesse curso foram <h1> até <h6>, <p>, <mark>, <small>, <i>, <u>, <strong>, <ol>, <ul>, <li>, <a>, <hr>, <sub>, <sup>, <blockquote>, <input>, entre outras.

Atributos das Tags

Os atributos das tags HTML fornecem informações adicionais sobre os elementos HTML. Eles são sempre especificados na tag inicial e geralmente vêm em pares de nome/valor, como nome="valor"
